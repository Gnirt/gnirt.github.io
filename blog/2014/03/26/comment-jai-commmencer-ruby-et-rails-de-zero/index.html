Comment J’ai Commmencer Ruby Et Rails De Zéro?

Mar 26th, 2014 | Comments

J’ai fais la découverte de Ruby a la fin d’année 2013, suite aux nombreux avis de la communauté Ruby (Rencontre à Paris.rb, différents développeurs rencontrés inopinément). Ayant seulement fait du développement web sur du php auparavant. J’ai voulu m’essayer à un langage de programmation orientée objet. J’avais donc (beaucoup?) entendu parler de Ruby, j’ai commencé à regarder à quoi ça ressemblait.

Ressources ruby :

J’ai commencer avec Ruby Monk ce site permet d’apprendre les bases de Ruby sans aucune installation sur son ordinateur sous forme de leçon.

Les autres ressources qui on aussi bonne réputation :

Try Ruby
Ruby Koans
Learn code the hard way

Référence spécial pour Ruby Warrior qui permet d’apprendre le ruby sous la forme d’un jeu (en plus le design en pixel art est magnifique !)

Ensuite rails :

J’ai pour ma part commencer avec Rails Tutorial qui vous fait coder une version basique de l’application Twitter. Au passage en plus de vous apprendre rails et il vous apprend toute la partie déploiement de l’application ! (Git, Heroku..)

Mais aussi :

Rails for Zombies pour apprendre de façon ludique !
les guides officielles
Rails Casts qui propose des vidéos succinctes sur des sujets en particulier

Ruby Gems pour toutes les gems
Ruby Toolbox pour toutes les gems

et beaucoup de stackoverflow :)
