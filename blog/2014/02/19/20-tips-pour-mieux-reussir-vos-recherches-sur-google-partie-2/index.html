20 Conseils Pour Mieux Réussir Vos Recherches Sur Google (Partie 2)

Feb 19th, 2014 | Comments

Aujourd’hui je partage avec vous 10 autres tips pour faire des recherches plus efficaces sur Google (ou comme j’aime l’appeler l’Oracle) ! Vous pouvez retrouver la première partie ici

  1. Lien
    Le symbole “link:” trouveras les pages contenant un lien vers une URL

    ex : link: philippetring.fr

  2. Recherche Vertical
    Plutot que de faire une recherche à travers tout le web, google met a notre disposition plusieurs moteurs plus spécifique.

    • http://blogsearch.google.com/
    • http://books.google.com/
    • http://scholar.google.com/
    • http://catalogs.google.com/
    • http://www.google.com/codesearch
    • http://www.google.com/dirhp
    • http://finance.google.com/finance
    • http://images.google.com/
    • http://maps.google.com/maps
    • http://news.google.com/
    • http://www.google.com/patents
    • http://www.google.com/products
    • http://video.google.com/

  3. Films
    “Film:” permet de lister les cinémas les plus proches et les horaires si il est suivi d’un code postal, d’une ville etc..

  4. Musique
    Musique: pour le contenu relatif a de la musique

  5. Conversion d’unité
    Unité de distance, de monnaie

    ex : 12 meters in miles

  6. Nombre particulier
    Dates, code postaux, numéro de téléphone, Google reconnait différents pattern.

  7. Type de fichier
    “filetype:” pour lister uniquement les fichiers d’un type (pdf, word, etc..)

  8. Recherche sur tag
    Si vous souhaitez baser votre recherche sur le contenu html vous pouvez utiliser : “inurl:”, “intitle:”, “intext:”, and “inanchor:”

  9. Cache
    Cette fonctionnalitée peu connue permet de faire une recherche sur le cache de google (donc retrouver des pages possiblement supprimées)

  10. Answer to life, the universe, and everything
    L’oracle vous répondras.
